# crxlsxwriter

Crystal bindings to the libxlsxwriter library.

## Installation

1. Add the dependency to your `shard.yml`:

   ```yaml
   dependencies:
     crxlsxwriter:
       github: Rolltrax/crxlsxwriter
   ```

2. Run `shards install`

## Usage

```crystal
require "crxlsxwriter"
```
